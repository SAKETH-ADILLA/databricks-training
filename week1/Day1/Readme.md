# Day 1 - SQL Practice (Databricks Training)

## 📌 Overview

This is my Day 1 SQL practice as part of my Databricks Data Engineering learning journey.

On Day 1, I focused on building strong SQL fundamentals using MySQL by working with Employee, Department, and Project tables.

The goal was to understand how SQL works in real-world scenarios using basic queries, filtering, aggregate functions, grouping, and sorting.

## 📚 Topics Learned in Day 1

### Basic SELECT Queries

* Retrieve all employee records
* Retrieve selected columns
* Retrieve department names

### WHERE Clause Filtering

* Employees older than 30
* Employees from IT department
* Employees based on department ID

### LIKE Operator Practice

* Names starting with J
* Names ending with e
* Names containing specific characters
* Name length conditions

### Date Functions

* Employees hired in 2020
* Employees hired in January
* Employees hired before 2019
* Employees hired after a specific date

### Aggregate Functions

* Total Salary using SUM()
* Average Salary using AVG()
* Minimum Salary using MIN()
* Maximum Salary using MAX()

### GROUP BY + HAVING

* Employee count by department
* Average salary by department
* Total salary by department
* Departments with more than 2 employees
* Departments with high average salary

### ORDER BY

* Sorting by salary
* Sorting by age
* Sorting by hire date
* Sorting by department
