# Day 3 - SQL Joins Assignment

## 📌 Overview

Day 3 focused on SQL JOIN operations and understanding how unmatched rows are handled using different types of joins.

This assignment was based on a simple online learning platform database using:

* students
* courses
* enrollments
* instructors

The main focus was on solving real-world SQL problems using:

* LEFT JOIN
* RIGHT JOIN
* FULL OUTER JOIN logic (using UNION in MySQL)
* CROSS JOIN

Since MySQL does not support FULL OUTER JOIN directly, it was implemented using LEFT JOIN + RIGHT JOIN + UNION.
