# Day 4 - Window Functions + CTEs + Advanced SQL Analytics

## 📌 Overview

Day 4 focused on advanced SQL analytical concepts using Window Functions and Common Table Expressions (CTEs).

This day was based on a retail sales company dataset containing:

* employees
* customers
* orders

The main goal was to understand how analytical SQL works in real-world reporting, ranking, cumulative calculations, and hierarchy-based problems.

This is where SQL starts becoming powerful for business analysis instead of just simple querying.
